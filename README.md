# ActiSight – Activity Tracking & Analysis System

ActiSight is a web-based application that tracks and analyzes human activity data using a Python (Flask) backend and an HTML/CSS frontend. The system processes structured activity logs and provides insights such as time spent, activity type, and user-wise behavior patterns.

## 🚀 Features

* 🔐 User Login System (Flask + SQLite)
* 📊 Activity Tracking from CSV Dataset
* 🔍 Search by Name and Time Range
* ⏱️ Total Time Calculation per Person
* 📈 Data Visualization using Chart.js
* 🖥️ Simple and Clean HTML/CSS Interface

## 🧱 Tech Stack

**Frontend:**

* HTML
* CSS
* Chart.js

**Backend:**

* Python (Flask)

**Database:**

* SQLite (for user authentication)

**Data Processing:**

* Pandas (CSV handling and analytics)

## ⚙️ How It Works

1. User logs into the system via a simple authentication interface
2. Backend reads activity data from a CSV file
3. Data is processed using Pandas to calculate durations and filters
4. Users can search records by name or time range
5. Results are displayed in tables and charts on the dashboard

## 📊 Use Cases

* Basic activity monitoring systems
* Student-level data analysis projects
* Time tracking and reporting tools

## 📌 Status

✔ Functional prototype using Flask and static frontend
🚧 Can be upgraded to full-stack system with React and PostgreSQL
